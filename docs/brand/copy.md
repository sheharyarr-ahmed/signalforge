## Longer description (HF Space card / README hero)
SignalForge is a production-grade NLP inference microservice built with FastAPI, serving open-source Hugging Face models through a clean REST API. It provides multi-language sentiment analysis (with an API-to-local fallback path), zero-shot classification against user-defined labels, local 384-dimensional embeddings, and pgvector-backed semantic search — all on CPU-only, free-tier infrastructure. Every response reports which model served it, its latency, and whether a fallback was triggered.
