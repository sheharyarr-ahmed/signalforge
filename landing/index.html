API surface

Endpoints

Strict Pydantic v2 at every boundary. Four-class Hugging Face error handling — no generic catch-all.

MethodPathDoes
GET/healthZero-dependency liveness check.
POST/api/v1/sentimentMulti-language sentiment; HF API with local CPU fallback.
POST/api/v1/classifyZero-shot against caller-supplied labels (API-only).
POST/api/v1/embedLocal 384-dim embeddings; stored in pgvector.
POST/api/v1/searchSemantic search over stored documents.
Quickstart

One curl away

Every inference response reports its own provenance.

# Multi-language sentiment
curl -s -X POST https://sheryyahmed457-signalforge.hf.space/api/v1/sentiment \
  -H 'Content-Type: application/json' \
  -d '{"text":"Este producto es increíble"}'

# → response
{
  "label": "positive",
  "inference_metadata": {
    "model": "cardiffnlp/twitter-xlm-roberta-base-sentiment",
    "provider": "huggingface_api",
    "processing_time_ms": 412,
    "fallback_triggered": false,
    "confidence": 0.97
  }
}
🛰️

Runs on free-tier infrastructure. The Hugging Face Space sleeps after inactivity (~1 min cold wake) and Supabase pauses after ~7 idle days. This is a portfolio artifact demonstrating open-source ML deployment discipline — not a high-availability production endpoint. The repo's SCALING.md documents every free-tier limit and its paid upgrade path.
